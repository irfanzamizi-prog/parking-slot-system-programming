# 🅿️ Parking Slot Reservation System

A command-line application for managing and reserving parking slots,
built as part of my Computer Science Diploma coursework at 
Kolej Profesional MARA (KPM) Indera Mahkota.

---

## 📋 What It Does

- View all available and occupied parking slots in real time
- Reserve a parking slot by entering vehicle details
- Cancel an existing reservation
- Data is saved to a file so reservations persist between sessions
- Input validation and error handling throughout

---

## 🛠️ Built With

- **Language:** Python 3
- **Concepts used:** String manipulation, file handling, exception handling
- **Storage:** Text file (reservations.txt)

---

## 🚀 How to Run

1. Make sure Python 3 is installed on your computer
2. Clone or download this repository
3. Open terminal / command prompt in the project folder
4. Run:
```
python main.py
```
